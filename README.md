# Niko-py
基于Nonebot2开发的QQ机器人

## 已实现功能
- [x] 一言
- [x] bing每日一图
- [x] 来点setu
- [x] 喜报图片生成 
- [x] 微博订阅
- [x] 天气查询（待完善）
- [x] 基于 [强智教务API](https://github.com/WindrunnerMax/SHST) 的课表查询（待完善）
- [x] 网易云音乐点歌（待完善）
- [x] ...

## 依赖插件
- nonebot-plugin-htmlrender 

```bash
nb plugin install nonebot-plugin-htmlrender
```
